# NVIDIA RAPIDS (cuDF / cuML)
Aceleração de pipelines de dados e ML em GPU. cuDF = ETL/dataframes em GPU (API estilo pandas); cuML = machine learning clássico acelerado. Compatível com Dask para escala multi-GPU/multi-nó. Ideal para startups que processam grandes volumes de dados tabulares.
Fonte: https://rapids.ai/
