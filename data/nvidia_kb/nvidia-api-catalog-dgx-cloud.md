# NVIDIA API Catalog e DGX Cloud

## O que é
- **NVIDIA API Catalog (build.nvidia.com)** — catálogo onde é possível **experimentar centenas
  de modelos** (LLMs, visão, fala, embeddings, rerankers) via endpoints hospedados e, quando a
  startup decide self-hostar, baixar o mesmo modelo como **NIM**. É o on-ramp de menor atrito.
- **DGX Cloud** — infraestrutura de **GPU gerenciada** para treino e inferência em larga escala,
  acessível inclusive via créditos do **NVIDIA Inception**.

## Por que importa para startups AI-native
Reduz a barreira de entrada: a startup testa rápido no API Catalog (sem comprar GPU) e escala
para DGX Cloud quando precisa de treino pesado ou throughput de produção — caminho natural de
"API externa → infraestrutura própria otimizada".

## Quando recomendar (sinais na startup)
- Quer **avaliar/prototipar** modelos antes de investir em infraestrutura.
- Precisa de **GPU sob demanda** para treino/fine-tuning sem capex.
- É elegível a **créditos Inception** para compute.
