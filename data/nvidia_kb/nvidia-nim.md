# NVIDIA NIM
Microservices de inferência prontos para deploy de modelos de IA otimizados. Um NIM inspeciona formato/arquitetura/quantização do modelo e escolhe automaticamente o runtime ótimo (vLLM, SGLang ou TensorRT-LLM). Integra com Triton/Dynamo. Reduz custo e latência frente a APIs externas genéricas; permite self-hosting e governança. Ideal para startups que dependem só de API externa e precisam internalizar inferência.
Fonte: https://www.nvidia.com/en-us/ai-data-science/products/nim-microservices/
