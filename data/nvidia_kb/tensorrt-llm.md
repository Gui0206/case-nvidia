# TensorRT-LLM
Biblioteca de otimização de inferência de LLMs em GPUs NVIDIA. API Python para definir modelos com otimizações SOTA (quantização, kernels fundidos, in-flight batching), gerando runtimes de alta performance. Reduz custo por token e latência p95. Recomendado para produtos sensíveis a latência e para reduzir custo de inferência de modelos 7B–405B.
Fonte: https://github.com/NVIDIA/TensorRT-LLM
